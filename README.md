## Local Development

1. Create conda environment. For example:

```shell
conda create -n pymmmc_env python=3.8
```

2. Activate environment.

```shell
conda activate pymmmc_env
```

3. Install requirements:

```shell
make init
```

4. To run tests:

```pyhton
make test
```

5. Set [pre-commit hooks](https://pre-commit.com/) (Optional):

```shell
pre-commit install
```
